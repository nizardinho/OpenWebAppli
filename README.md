# OpenWebAppli

Web application with user logging and addition of one or more boat in a data base. 

- Creation and logging User
- Insert Boat and User into data Base 

Tecnologies
 - Spring Boot 1.4.4 
 - Spring security
 - HSQL 2.3.4 
 - JPA 
 - Tomcat container
 - Maven 3


Prerequisites
    Java 8+
    Maven 3
